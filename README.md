# Install

## Windows
`xcopy qt-dark.xml %APPDATA%\QtProject\qtcreator\styles`

## MacOS
`cp qt-dark.xml ~/.config/QtProject/qtcreator/styles/`

## Linux
`cp qt-dark.xml ~/.config/QtProject/qtcreator/styles/`
